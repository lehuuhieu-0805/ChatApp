## Install

#### Clone project

`git clone https://github.com/lehuuhieu-1011/ChatApp.git`

`cd ChatApp`

`cd clients`

#### Install dependencies

`npm install`

## Usage

In root directory, run Server

`cd server`

`dotnet run`

In root directory, run UI

`cd clients`

`npm start`
